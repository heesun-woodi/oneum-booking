# 온음 예약 시스템 요구사항 v2

## 사용자 시나리오

### 1. 회원 (온음 세대)
1. 메인 화면 "세대 전용" 버튼 클릭
2. 로그인 또는 회원가입
   - 회원가입: 세대 선택 + 이름 + 전화번호 (필수)
3. 로그인 후 → 회원 전용 예약 UI
4. 달력에서 날짜 선택 → 연속 시간대 선택 가능
5. 예약 완료 → 달력에 예약 표시
6. "예약 변경/취소" → 로그인 필요 → 본인 예약만 수정 가능

### 2. 비회원
1. 메인 화면에서 바로 날짜 클릭 (로그인 없이)
2. 비회원 예약 UI
3. 연속 시간대 선택 + 이름 + 전화번호 입력
4. 예약 완료 → 달력에 예약 표시
5. "예약 변경/취소" → 전화번호로 조회 → 수정 가능

---

## 메인 화면 구성

### 상단 버튼
- [ ] **세대 전용** 버튼 (로그인/회원가입)
- [ ] **예약 변경/취소** 버튼

### 달력
- [ ] 월 네비게이션 (이전/다음 달)
- [ ] 현재 월 표시 (예: 2026년 3월)
- [ ] 예약된 시간 시각적 표시 (색상/뱃지)

### 공간 선택 탭
- [ ] 놀터 탭 (동작)
- [ ] 방음실 탭 (동작)

---

## 예약 모달 (날짜 클릭 시)

### 회원 로그인 상태
- [ ] 선택한 날짜 표시
- [ ] 연속 시간대 선택 (다중 선택 가능)
  - 예: 14:00, 15:00, 16:00 선택 → 3시간 예약
- [ ] 공간 선택 (놀터/방음실)
- [ ] 예약자 정보 (자동 입력, 수정 가능)
- [ ] 예약하기 버튼

### 비회원 상태
- [ ] 선택한 날짜 표시
- [ ] 연속 시간대 선택
- [ ] 공간 선택
- [ ] 예약자 정보 입력 (이름, 전화번호)
- [ ] 예약하기 버튼

---

## 로그인/회원가입 모달

### 회원가입
- [ ] 세대 선택 (201, 301, 302, 401, 402, 501)
- [ ] 이름 입력
- [ ] 전화번호 입력
- [ ] 비밀번호 설정
- [ ] 가입하기 버튼

### 로그인
- [ ] 세대 선택
- [ ] 비밀번호 입력
- [ ] 로그인 버튼

---

## 예약 변경/취소 모달

### 기본 (비회원 조회)
- [ ] 전화번호 입력
- [ ] 조회하기 → 예약 목록 표시
- [ ] 수정/취소 버튼

### 세대 로그인
- [ ] "세대로그인" 버튼
- [ ] 로그인 후 본인 예약 자동 표시
- [ ] 수정/취소 버튼

## 달력 표시

### 예약 상태 시각화
- [ ] 예약 없음: 흰색
- [ ] 일부 예약: 노란색 (시간대별 표시)
- [ ] 전체 예약: 회색 (예약 불가)

### 월 네비게이션
- [ ] 이전 달 버튼
- [ ] 다음 달 버튼
- [ ] 현재 월 표시
