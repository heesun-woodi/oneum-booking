## 🔢 메시지 타입별 분류

### 회원가입 (1-X)
- **1-2**: 가입 승인 → 신규 회원
- **1-3**: 가입 거부 → 신청자

### 예약 (2-X)
- **2-1**: 예약 완료 → 회원
- **2-2**: 예약 완료 + 입금 안내 → 비회원
- **2-3**: 예약 취소 → 예약자 (입금 완료자만)

### 입금 (3-X)
- **3-1**: 입금 확인 → 비회원
- **3-2**: 입금 리마인더 → 미입금 비회원 (D-7, D-5, D-2)

### 리마인더 (4-X)
- **4-1**: 내일 예약 안내 → 비회원 (입금 완료)
- **4-2**: 오늘 예약 안내 → 비회원 (계절별)
- **4-3**: 내일 예약 안내 → 회원

### 재무 (5-X)
- **5-2**: 미입금 알림 → 재무담당자
- **5-3**: 환불 안내 → 재무담당자

### 관리자 (6-X)
- **6-1**: 회원가입 신청 → 관리자
