## 🔧 템플릿 등록 절차

### 1. SOLAPI 관리자 페이지 접속
- https://console.solapi.com
- 로그인 후 [알림톡] → [템플릿 관리] 메뉴 이동

### 2. 템플릿 등록
1. **템플릿 추가** 버튼 클릭
2. **템플릿 코드** 입력 (예: `oneum_signup_approved`)
3. **템플릿 내용** 복사/붙여넣기
4. **변수 설정** (#{변수명} 형식으로 자동 인식됨)
5. **저장** 후 **승인 요청**

### 3. 승인 대기
- 카카오 심사팀이 1-2일 내 검토
- 승인 완료 시 이메일/SMS 알림

### 4. 템플릿 ID 확인 및 코드 반영
- 승인된 템플릿의 **템플릿 ID** 확인
- `lib/notifications/alimtalk-templates.ts` 파일에서 `ALIMTALK_TEMPLATE_IDS` 수정

**예시**:
```typescript
export const ALIMTALK_TEMPLATE_IDS: Record<MessageType, string> = {
  '1-2': 'oneum_signup_approved',      // ← 실제 승인된 템플릿 코드로 교체
  '1-3': 'oneum_signup_rejected',
  // ...
}
```

## ⚠️ 주의사항

### 템플릿 작성 규칙
1. **변수 형식**: 반드시 `#{변수명}` 형식 사용
2. **글자 수 제한**: 템플릿당 최대 1,000자
3. **버튼**: 최대 5개까지 추가 가능
4. **이모지**: 사용 가능하지만 일부 특수 이모지는 제한될 수 있음

### 승인 거부 사유
- 광고성 문구 포함
- 불명확한 발신자 정보
- 변수 사용 오류
- 카카오 정책 위반

### 변경 사항 발생 시
- 템플릿 내용 수정 시 **재심사 필요**
- 변수 추가/삭제 시에도 **재승인 필요**
